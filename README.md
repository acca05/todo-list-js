# Lista de Tarefas – JavaScript

Projeto simples de lista de tarefas desenvolvido em JavaScript puro, com foco em praticar lógica de programação, manipulação do DOM e armazenamento local.

## Funcionalidades
- Adicionar tarefas
- Marcar tarefas como concluídas
- Excluir tarefas
- Salvar tarefas no navegador (localStorage)

## Tecnologias utilizadas
- HTML
- CSS3
- JavaScript (Vanilla)
- LocalStorage

## Como executar o projeto
1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Comece a adicionar suas tarefas

## Aprendizados
- Manipulação do DOM
- Eventos em JavaScript
- Funções
- Uso do localStorage
